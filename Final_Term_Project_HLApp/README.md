# Healthcare Appointment System

Full-stack web application for secure online doctor appointment booking and continuous patient treatment tracking.

## Stack

- Frontend: Next.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Notifications: Email and mobile alerts

## Core Modules

- Patient registration, login, and profile management
- Doctor management by administrators
- Appointment booking, confirmation, and doctor assignment
- Treatment records and physical checkup tracking
- Prescription and medication schedule management
- Follow-up visit scheduling
- Automated appointment and medication notifications
